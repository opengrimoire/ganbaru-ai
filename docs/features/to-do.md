# To-do

The To-do section is reserved for a future task layer. The current app keeps the tab in the title bar so navigation has a stable home for that work, but the content area is intentionally empty until the new design is settled.

## Current status

The previous board implementation, task table, task store, and task priority theme tokens have been removed. The future To-do design should start from the product goals rather than inheriting the old columns, priorities, or card controls.
